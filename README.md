⚙️ Installation & Setup Guide

🔹 Prerequisites

Make sure you have installed:

- Node.js (v18+)
- XAMPP (Apache + MySQL)
- MongoDB (or MongoDB Atlas)
- VS Code

---

▶️ 1. Run Portfolio Website

cd portfolio
npm install
npm run dev

Open:

http://localhost:5173

---

▶️ 2. Run Brew Haven Café Website

Step 1 — Move files

Place the folder inside:

C:\xampp\htdocs\brewhaven

Step 2 — Start XAMPP

- Start Apache ✅
- Start MySQL ✅

Step 3 — Import Database

- Open: http://localhost/phpmyadmin
- Import "brewhaven.sql"

Step 4 — Run Website

http://localhost/brewhaven/index.html

---

▶️ 3. Run CRM System

Backend

cd crm/backend
npm install
cp .env.example .env
node seed.js
npm run dev

Frontend

cd crm/frontend
npm install
npm start

Open:

http://localhost:3000
